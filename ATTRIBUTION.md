# Attribution

Timing, displacement, and rotation values for route and feather presets are derived from the
[Windows Phone Toolkit](https://github.com/MicrosoftArchive/WindowsPhoneToolkit) transition storyboards
(Ms-PL) and `TurnstileFeatherEffect` constants, plus WinUI `ProgressRing` timing where noted in tokens.

This package does not redistribute the Toolkit assemblies or XAML at runtime; it provides
CSS equivalents for web projects.
